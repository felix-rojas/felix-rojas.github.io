  • ¿Cuál es la diferencia entre Internet y la World Wide Web?

    • La WWW se conoce como la web y es un sistema de información, mientras que la Internet es una red de computadoras que interconecta a los dispositivos físicos de manera global.

  • ¿Cuáles son las partes de un URL?

    • - La primera parte es el protocolo a utilizar para la transferencia de datos. Se le denomina scheme.
    - La segunda parte es el subdominio de la URL como "www", "blog", etc. La más común es "www".
    - La tercera parte es el dominio de la URL. Lo que usualmente describen las personas cuando se refieren a un sitio web.
    - La cuarta parte le denominan el dominio de nivel tope. Es usualmente del país y/o compañía a la que pertenece el dominio
    - La última parte es usualmente la ubicación de la página web a cargar y se denomina path. Por ejemplo: google.com /search
    - En algunos sitios podemos encontrar query strings que son peticiones a la página web. Al pulsar el link inferior, se abrirá una búsqueda en google buscando "cute dogs"
    https://www.google.com/search?q=cute+dogs

  • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

    • GET se utiliza para pedir datos y contienen un cuerpo de respuesta

    HEAD se usa para pedir datos pero no contiene un cuerpo de respuesta

    POST manda datos al servidor. Por ejemplo, al llenar una encuesta con el semantic header form. Los manda sin encriptar y puede ser texto plano, imgs, etc. POST es el único método que siempre es no idempotente, pues cambia el estado del servidor.

    PUT hace lo mismo que POST pero es siempre idempotente. Llamarlo varias veces no altera el resultado ni el estdo del servidor

    PATCH es un conjunto de instrucciones, que a diferencia del PUT, no se obtiene un cuerpo de respuesta
